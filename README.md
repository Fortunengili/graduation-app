# Graduation App
This is an interactive app that incorporated a JavaScript library known as React and was styled using tailwind.
The app was meant to showcase personal touches such as showcasing its owner graduated with a degree in construction.
The app showcased a well decorated hero-landing page followed by the days program.
# Features
The app had a landing page followed by a page showcasing the days program. 
It had links at the bottom redirecting to the spotify playlist, the location and an RSVP form.
# Steps to Install:
The first command in installing react was
1. npm create vite@latest graduation-app
2. cd graduation-app
3. npm install- installing dependencies
4. npm run dev- to run the app in a localhost
